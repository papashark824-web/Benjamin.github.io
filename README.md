# Benjamin.github.io

This is my personal GitHub Pages website featuring a Bohol Travel Portfolio.

## About

This website showcases the beauty of Bohol, Philippines - famous for its Chocolate Hills, pristine beaches, and unique wildlife like the Philippine Tarsier. The site includes:

- Welcome section with stunning images
- About section sharing my travel experiences
- Responsive design with a beautiful gradient background

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid and Flexbox)
- Hosted on GitHub Pages
